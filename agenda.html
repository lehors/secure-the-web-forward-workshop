Live sessions

Three live sessions are scheduled on 26-27-28 September 2023 to discuss selected position papers and converge on a common understanding of whether and how things may progress from a standardization perspective.

The calls are open to all invited workshop participants. Please register to attend the live sessions. If you registered already but haven't been notified of the live sessions, please contact Fran√ßois Daoust <fd@w3.org>.

Familiarity with selected position papers is recommended. Presentations of the papers during the live sessions will be short. We will rather ask participants to take an active role and help shape next steps for each of the topics.

To help guide discussions, the program committee encourages you to capture needs, questions and suggestions for standardization work linked to the topics addressed in the position papers before the live sessions through issues in the w3c/secure-the-web-workshop GitHub repository

Live session 1: Supply Chain Security

When: Tuesday 26 September 2023 - 3pm UTC - 2h (see local time conversion)

  • Context for the workshop
    • Securing software development and deployment platforms
    • Results of the Web Security Developer Experience Short Survey run on MDN in May 2023
  • Session intro - Setting the context
  • Review papers and discuss:
    • Software Bill of Materials for web frontends (Jan Kowalleck)
    • Establish Standards to Support Web Access to SBOM Data (Gary O'Neall)
    • Source Code Transparency (Daniel Huigens)
  • Next steps
